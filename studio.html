Setup & troubleshooting

Start ComfyUI on the host so this phone can reach it over Tailscale:

python main.py --listen --enable-cors-header
  • --listen binds ComfyUI to your tailnet (not just localhost).
  • --enable-cors-header lets this app talk to it.

Don’t open this from an email or Files attachment. Android runs it sandboxed (the address bar shows content://… or file://…) and blocks it from reaching your server.

Best way on a phone — serve it from the ComfyUI host. Put studio.html in a folder on that machine and run:

python -m http.server 8189

Then open http://100.116.157.100:8189/studio.html in Chrome. The page now lives on your tailnet, so if it loads it can reach the API.

Either way, ComfyUI must be started with both flags, and your phone must be on the same Tailscale network. The address above is your Tailscale IP and port, e.g. http://100.116.157.100:8188.

Stuck? Open http://100.116.157.100:8188 in a plain browser tab. If that won’t load, it’s Tailscale or the server — not this app.

Add to your Home Screen: iPhone → Share → Add to Home Screen. Android Chrome → ⋮ → Add to Home screen. It then opens full-screen like an app.

Z-Image runs out of the box. The other workflows run after you import their API JSON once (the button on each tab).
